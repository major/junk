Data migration solution from on-premise or public cloud to OpenStack.
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**Abstract:**

Enterprises wanting to use OpenStack in their private cloud environments face the challenge of migrating their current data to OpenStack.
